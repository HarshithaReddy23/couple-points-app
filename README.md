# 💕 Couple Points App

A beautiful web application for couples to share and track love points with each other. Give points for sweet gestures, and use them for special rewards!

## ✨ Features

- **Separate User Logins** - Each partner has their own account
- **Shared Couple Dashboard** - Connect via a unique couple code
- **Give Points (1-5)** - Reward your partner with points and reasons
- **Use Points** - Redeem points for special treats and rewards
- **Points History** - Track all point exchanges and redemptions
- **User Profiles** - View statistics of points given and received
- **Responsive Design** - Works perfectly on all devices
- **Modern UI** - Beautiful gradients and smooth animations

## 🚀 How to Use

### For Couples:
1. **First Partner**: Register with username, password, and create a couple code
2. **Second Partner**: Register using the same couple code to join
3. **Start Sharing**: Give points for sweet gestures, help, or surprises
4. **Redeem Points**: Use accumulated points for special treats!

### Point Values:
- **Give Points**: +1 to +5 (for appreciating your partner)
- **Use Points**: -1, -5, -10, -20, -50 (for rewards like pizza 🍕, movies 🎬, gifts 🎁, spa 💆‍♀️, roses 🌹)

## 🛠️ Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with animations
- **Vanilla JavaScript** - No frameworks required
- **Local Storage** - Data persistence on client-side

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/couple-points-app.git
```

2. Navigate to the project directory:
```bash
cd couple-points-app
```

3. Open `index.html` in your browser or serve with a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Or use any live server extension in your code editor
```

## 🔒 Privacy & Security

- All data is stored locally in your browser
- No data is sent to external servers
- Couple codes are private and only known to partners
- Clear your data anytime by clearing browser storage

## 🎨 Customization

### Change Colors:
Edit `styles.css` and modify the gradient values:
```css
background: linear-gradient(135deg, #your-color-1, #your-color-2);
```

### Modify Point Values:
Edit `index.html` and update the `data-points` attributes in the point options.

### Add Custom Rewards:
Modify the emoji and labels in the "Use Points" section to match your couple's preferences.

## 🐛 Troubleshooting

- **Points not saving?** Make sure cookies and local storage are enabled in your browser
- **Can't see partner?** Ensure both partners use the exact same couple code
- **Lost couple code?** Check the profile tab to see your current couple code
